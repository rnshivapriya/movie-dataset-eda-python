# 🎬 Movie Dataset Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a movie dataset using Python. The objective is to clean the data, analyze patterns, create visualizations, and derive meaningful insights.

---

## 🛠️ Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset

- Movie Dataset
- ~9,800 records
- Includes genres, ratings, popularity, release dates, languages, and other movie information.

---

## 📊 Workflow

1. Import Dataset
2. Data Cleaning
3. Handle Missing Values
4. Remove Duplicates
5. Exploratory Data Analysis (EDA)
6. Data Visualization
7. Generate Insights

---

## 📈 Key Insights

- Identified popular movie genres.
- Analyzed movie ratings and popularity.
- Explored release trends over time.
- Visualized language distribution.
- Generated insights from cleaned data.
